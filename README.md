ember-seisplotjs
==============================================================================

Wrapping [seisplotjs](https://github.com/crotwell/seisplotjs) for use in
[Ember](http://emberjs.com).


Compatibility
------------------------------------------------------------------------------

* Ember.js v2.18 or above
* Ember CLI v2.13 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-seisplotjs
```
